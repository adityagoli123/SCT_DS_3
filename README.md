# IBM HR Analytics Attrition Dashboard

Interactive HR Analytics dashboard created using Tableau Public.

## Project Objective
Analyze employee attrition patterns and identify factors contributing to workforce turnover.

## Key Insights
- Employees aged 25–34 have highest attrition
- Overtime employees leave more frequently
- Sales and R&D departments show higher attrition
- Lower job satisfaction correlates with attrition

## Tools Used
- Tableau Public
- HR Analytics Dataset
- Data Visualization
- Dashboard Design

## Dashboard Features
- KPI Cards
- Interactive Filters
- Treemap Visualization
- Overtime Analysis
- Attrition Analysis by Gender, Department, and Age Group
